# Dashboard 四窗格覆蓋設計

## 目標
- 直接覆蓋現有首頁 `Dashboard`
- 視覺方向改為類似桌面券商終端的四窗格密集布局
- 保留現有資料來源、互動邏輯與左側市場內容
- 不改後端、不新增資料流

## 版面
- 左上：市場窗格
  - 預設全市場清單
  - 可切換為類股熱度排行
  - 內部卷動，不把整頁撐長
- 右上：主圖窗格
  - 單一標的主圖
  - 即時 / 歷史切換
  - 技術線按鈕
- 左下：交易窗格
  - 成交 / 平倉時間線
  - 目前持倉摘要
- 右下：資訊窗格
  - 標的摘要
  - 價格結構
  - 技術位階
  - 帳本摘要

## 原則
- 不刪除 Python 或 HTML 檔案，因為前端實際由 `tsx` 元件組成
- 以 `Dashboard.tsx` 為首頁覆蓋點
- `AppShell.tsx` 同步整理文案，避免首頁改好但殼層仍是亂碼
- 整體頁面固定在單屏內，改由窗格內部卷動

## 驗收
- 首頁可見四個清楚窗格：市場、主圖、交易、資訊
- 左上窗格可在全市場清單與類股熱度間切換
- 右上主圖維持原有圖表功能
- 下排不再是長頁式排版
- 文案為乾淨中文
